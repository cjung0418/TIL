고지식한 알고리즘(Brute Force): 

```python
본문 문자열을 처음부터 끝까지 차례대로 순회하면서 일일이 비교
검색알고리즘:
def BruteForce(p,t):
    i=0 # t의 인덱스
    j=0 # p의 인덱스
    while j<M and i<N:
        if t[i] != p[j]:
            i=i+1
            j=j+1
    if j==M: return i-M #검색 성공
    else: return -1 #검색 실패
(단, p=찾을 패턴, t=전체 텍스트, M=len(p) #찾을 패턴의 길이, N=len(t) #전체 텍스트의 길이)
알고리즘의 시간 복잡도:O(MN)
```

KMP알고리즘:

```
불일치가 발생한 텍스트 문자열의 앞 부분에 어떤 문자가 있는지를 미리 알고 있으므로, 불일치가 발생한 앞 부분에 대하여 다시 비교하지 않고 매칭 수행
1.패턴을 전처리하여 배열 next[M] (불일치가 발생했을 경우 이동할 다음 위치)을 구해서 잘못된 시작을 최소화함 ->시간 복잡도는 O(M+N) 
매칭이 실패했을 때 돌아갈 곳을 계산. ->돌아갈 곳을 미리 정해놓음
```

보이어-무어 알고리즘: 

```
오른쪽에서 왼쪽으로 비교, 대부분의 상용 소프트웨어에서 채택하고 있는 알고리즘
패턴에 오른쪽 끝에 있는 문자가 불일치하고, 이 문자가 패턴 내에 존재하지 않는 경우, 이동거리는 패턴의 길이 만큼이 됨(최선의 경우) 
오른쪽 끝에 있는 문자가 불일치하고 이 문자가 패턴내에 존재하는 경우 그 부분에 맞춘다. ex1) winter를 찾는데, onwinter의 경우, win이 일치하기 때문에 두 칸을 점프
ex2) 'a pattern matching algorigthm'이 주어지고 'rithm'을 찾는 경우 rithm string의 skip 배열은 ['m'=0,'h'=1,'t'=2,'i'=3,'r'=4,다른 모든 문자=5]이다. 여기에서 skip 배열이 뜻하는 것은 비교를 했을 때, 일치하지 않는 해당부분이 't'라면 2칸 skip, 일치하지 않는 부분이 'r'이면 4칸 skip해야 한다는 의미이다.
```

문자열 매칭 알고리즘 비교: 

```
찾고자 하는 문자열 패턴의 길이 m, 총 문자열 길이 n
고지식한 패턴 검색 알고리즘: O(mn)
카프-라빈 알고리즘: O(n)
KMP 알고리즘: O(n)
보이어-무어 알고리즘: 최악의 경우 O(mn)이지만, 일반적으로 O(n)보다 작음
```
